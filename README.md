# xkcd Password Generator

xkcdスタイルの覚えやすくて安全なパスフレーズを作成するシンプルなWebアプリです。

GitHub Pagesで動作します。  
https://sososoy2.github.io/password-generator/

## できること

- 英単語をランダムに組み合わせてパスフレーズ生成
- 暗号学的乱数（`crypto.getRandomValues`）使用
- 単語の重複なし
- 数字・記号の挿入（オプション）
- 大文字化（オプション）
- ワンクリックコピー
- ダークモード / ライトモード切替

## 仕組み

- 単語リスト（words.txt）からランダムに選ぶ
- シャッフルして重複を防ぐ
- 数字や記号をランダム位置に挿入
- 最後に結合してパスワード生成

## ファイル構成
/index.html  
/script.js  
/words.txt  

## 使い方

1. 開く
2. 単語数などを設定
3. 「生成」を押す
4. コピーする

以上

## 注意

- パスワードの安全性は100%保証されるものではありません。
- パスワードのみではなく、多要素認証も併用してください。
- 強度はあくまでも目安なので過信しすぎないこと。

## 使用しているもの
